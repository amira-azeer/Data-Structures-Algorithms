## 1️⃣ Arrays
| Operation | Average / Worst Case |
| --------- | -------------------- |
| Access    | O(1)                 |
| Search    | O(n)                 |
| Insertion | O(n)                 |
| Deletion  | O(n)                 |

## 2️⃣ Linked Lists (Singly / Doubly) 
| Operation | Singly | Doubly |
| --------- | ------ | ------ |
| Access    | O(n)   | O(n)   |
| Search    | O(n)   | O(n)   |
| Insertion | O(1)*  | O(1)*  |
| Deletion  | O(1)*  | O(1)*  |

## 3️⃣ Stacks & Queues
| Operation      | Time Complexity |
| -------------- | --------------- |
| Push / Enqueue | O(1)            |
| Pop / Dequeue  | O(1)            |
| Peek           | O(1)            |
| Search         | O(n)            |


## 4️⃣ Hash Tables / Maps
| Operation | Average | Worst Case |
| --------- | ------- | ---------- |
| Insert    | O(1)    | O(n)       |
| Search    | O(1)    | O(n)       |
| Delete    | O(1)    | O(n)       |


## 5️⃣ Trees
| Operation | BST Avg  | BST Worst | Balanced BST (AVL, Red-Black) |
| --------- | -------- | --------- | ----------------------------- |
| Search    | O(log n) | O(n)      | O(log n)                      |
| Insert    | O(log n) | O(n)      | O(log n)                      |
| Delete    | O(log n) | O(n)      | O(log n)                      |


## 6️⃣ Graphs
| Operation     | Adjacency List | Adjacency Matrix |
| ------------- | -------------- | ---------------- |
| Add vertex    | O(1)           | O(V^2)           |
| Add edge      | O(1)           | O(1)             |
| Remove vertex | O(V+E)         | O(V^2)           |
| Remove edge   | O(E)           | O(1)             |
| BFS / DFS     | O(V+E)         | O(V^2)           |


## 7️⃣ Common Algorithms
| Algorithm                                     | Time Complexity |
| --------------------------------------------- | --------------- |
| Bubble Sort / Insertion Sort / Selection Sort | O(n²)           |
| Merge Sort / Quick Sort                       | O(n log n)      |
| Linear Search                                 | O(n)            |
| Binary Search                                 | O(log n)        |
| DFS / BFS                                     | O(V+E)          |
| Dynamic Programming (typical)                 | O(n²) / O(n·m)  |
